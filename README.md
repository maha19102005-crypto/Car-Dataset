# Car Sales Analysis Dashboard

## Overview

This project presents an interactive Power BI dashboard for analyzing car sales data. It provides insights into vehicle models, prices, sales performance, engine specifications, and customer purchasing trends.

## Features

* Interactive KPI cards for sales and vehicle metrics.
* Visualizations of car models, prices, and sales trends.
* Slicers and filters for dynamic data exploration.
* Comparative analysis of vehicle performance and pricing.

## Tools Used

* Power BI
* Power Query
* DAX
* Car Sales Dataset

## Dashboard Insights

* Total cars and sales analysis
* Price distribution by model
* Engine power comparison
* Vehicle age and mileage analysis
* Sales trends and customer preferences

## Conclusion

This dashboard demonstrates data visualization and business intelligence techniques using Power BI to analyze car sales data and support data-driven decision-making.
